Section Number Editor

The Section Number Editor allows mass editing of course section numbers, subtitles, and designations. Simply make the changes you need and click OK. The program will check for duplications of section numbers on courses and display an error if there are any. This editor also has a Copy All button that will copy the contents of the grid as tab delimited text. If you do the copy before clicking OK you will have a complete copy of the changes that were made that can be loaded into most spreadsheets.
